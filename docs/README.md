## 定位

skilllock 是 [Agent Skills](https://agentskills.io) 的 **DevOps 层**：

| 工具 | 解决的问题 |
|------|-----------|
| APM / `skills` / skillpm | 怎么**装** skills |
| **skilllock** | 装完之后怎么**锁、验、diff、审计、复现、升级** |

示例项目：[`examples/demo-project`](../examples/demo-project/README.md)
