## Positioning

skilllock is the **DevOps layer** for [Agent Skills](https://agentskills.io):

| Tool | Problem |
|------|---------|
| APM / `skills` / skillpm | How to **install** skills |
| **skilllock** | How to **lock, verify, diff, audit, reproduce, and upgrade** after install |

Example project: [`examples/demo-project`](../examples/demo-project/README.md)
